# deskLight1_A
LED longboard light setup, by Thurstan

Repository is an Arduino project

### Equipment
#### Hardware
- Arduino Mini Pro board
  * 5v
  * ATmega328
  * 16 MHz clock speed
  * 1K EEPROM
- WS2812B addressable LED strips
  * pixel order = GRB
- Movement sensor
  
#### Software
- Arduino
- FastLED library  https://github.com/FastLED/FastLED
